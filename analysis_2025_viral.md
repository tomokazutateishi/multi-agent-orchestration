# 2025年バズったサービス分析と共通点・横展開

## 1. 2025年バズった主なサービス

| サービス | 概要・バズ要因 |
|----------|----------------|
| **ChatGPT** | 世界DL数1位（2025年3月月間4600万）。「AIといえばChatGPT」。買い物・料理・トリビア・ポップカルチャーなど日常生活用途に拡大。週末利用増。 |
| **Jiffcy** | テキスト通話アプリ。入力中の文字がリアルタイムで相手に表示。既読スルー・返信プレッシャーから解放。10代から大人・業務利用へ拡大。月間利用者前年比15倍。 |
| **RedNote（小紅書）** | TikTok代替として米国で急伸。ライフスタイル・レコメンド（美容・ファッション・旅行・食）。Instagram的UGC。 |
| **Meitu / EPIK** | 自然な画像加工。「ありのままの自分」を可愛く。加工より「自然さ」重視へシフト。 |
| **Gemini / DeepSeek** | 汎用AIアシスタント。ChatGPTに次ぐDL・利用。 |
| **健康・ウェルネスAI** | プロンプト成長カテゴリ1位。カロリーカウンター等、各カテゴリにAI組み込み。 |

## 2. 共通点の整理

1. **AIの日常化・多用途化** … チャットの枠を超え、買い物・料理・健康・仕事・ライフスタイルなど「その場で使うAI」が当たり前に。
2. **リアルタイム・低摩擦** … Jiffcy型「今すぐ見える／待たない」体験。既読プレッシャーからの解放。
3. **ライフスタイル・レコメンド** … RedNote型。縦型コンテンツ×美容・ファッション・食・旅行など「自分に合う情報」。
4. **自然さ・ありのまま** … Meitu型。過度な加工より「自分らしさを引き出す」。
5. **垂直統合×AI** … 健康・金融・仕事・教育など「分野特化＋AI」で200本超のアプリが機能追加。

## 3. 横展開の方向性（議論のたたき台）

- **リアルタイム協調** … Jiffcyの「見える化」を会議・ブレスト・学習に横展開（リアルタイムメモ、リアルタイム投票など）。
- **AI×生活垂直** … ChatGPTの「日常用途」を料理・ファッション・健康・旅行など1分野に特化して深く（レシピAI、コーデAI、健康記録AIなど）。
- **ライフスタイル・キュレーション** … RedNote型をニッチに（例：地域限定、趣味限定、職種限定の「おすすめ」プラットフォーム）。
- **自然出力・信頼感** … Meitu型をテキストに（「盛らない」要約・キャプション・自己紹介の生成）。
- **プラットフォーム代替・移行** … RedNoteのように「既存大手の代替」としてのポジション（プライバシー重視、広告なし、特定層向け）。

---

上記を踏まえ、社長・マーケター・プランナーで**横展開した優秀なサービスを5つ選び**、それぞれ完成までエンジニアと相談して進める。
