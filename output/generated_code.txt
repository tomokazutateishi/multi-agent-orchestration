まず、具体的な商品アイデア名と要件が入っていないため、仮の例として「タスク管理アプリ」を実装するものとして進めます。このMVPではユーザー登録・ログイン機能、タスクの追加・表示機能を実装します。技術スタックとしては、フロントエンドにReact、バックエンドにNode.jsとExpress、データベースにPostgreSQLを使用します。基礎的なUIは軽量なCSSフレームワーク（例: Bootstrap)を用います。

以下に、各要件に基づくソースコードを示します。

### 1. ユーザー登録・ログイン機能

#### 使用するライブラリ
- `bcrypt`（パスワードのハッシュ化）
- `jsonwebtoken`（JWTの生成）
- `pg`（PostgreSQLとの接続）

#### Expressによるバックエンド実装

```javascript
// server.js
const express = require('express');
const bodyParser = require('body-parser');
const jwt = require('jsonwebtoken');
const bcrypt = require('bcrypt');
const { Pool } = require('pg');

const app = express();
app.use(bodyParser.json());

const pool = new Pool({
  user: 'yourusername',
  host: 'localhost',
  database: 'task_manager',
  password: 'yourpassword',
  port: 5432,
});

const SECRET_KEY = 'your_jwt_secret_key';

app.post('/register', async (req, res) => {
  const { username, password } = req.body;
  const hashedPassword = await bcrypt.hash(password, 10);
  const result = await pool.query('INSERT INTO users (username, password) VALUES ($1, $2) RETURNING *', [username, hashedPassword]);
  res.json({ userId: result.rows[0].id });
});

app.post('/login', async (req, res) => {
  const { username, password } = req.body;
  const result = await pool.query('SELECT * FROM users WHERE username = $1', [username]);
  if (result.rows.length > 0) {
    const user = result.rows[0];
    if (await bcrypt.compare(password, user.password)) {
      const token = jwt.sign({ id: user.id }, SECRET_KEY);
      res.json({ token });
    } else {
      res.status(401).send('Unauthorized');
    }
  } else {
    res.status(401).send('Unauthorized');
  }
});

app.listen(5000, () => console.log('Server running on port 5000'));
```

### 2. 主要なデータ入力フォームとデータ表示機能

#### Reactによるフロントエンド実装

```javascript
// App.js
import React, { useState } from 'react';
import axios from 'axios';

const App = () => {
  const [tasks, setTasks] = useState([]);
  const [task, setTask] = useState('');

  const handleAddTask = async () => {
    // Assuming add task endpoint `/addTask` is implemented on server
    const response = await axios.post('http://localhost:5000/addTask', { task });
    setTasks([...tasks, response.data]);
    setTask('');
  };

  return (
    <div className="container">
      <h1>Task Manager</h1>
      <input value={task} onChange={(e) => setTask(e.target.value)} />
      <button onClick={handleAddTask}>Add Task</button>
      <ul>
        {tasks.map((task, index) => (
          <li key={index}>{task.name}</li>
        ))}
      </ul>
    </div>
  );
};

export default App;
```

### 3. デプロイ

開発環境では、ローカルでNode.jsサーバーを起動し、フロントエンドは`create-react-app`で起動します。以下が簡単な手順です。

#### フロントエンド
```sh
npx create-react-app task-manager
cd task-manager
npm start
```

#### バックエンド
```sh
node server.js
```
